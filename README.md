# baseAdmin

##### 本示例使用的公钥与私钥可自己生成，操作如下：
1. 打开命令行工具，输入openssl，打开openssl;
2. 生成私钥：genrsa -out rsa_private_key.pem 2048
3. 生成公钥： rsa -in rsa_private_key.pem -pubout -out rsa_public_key.pem
